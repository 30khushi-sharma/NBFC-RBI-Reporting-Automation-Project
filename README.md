1. Introduction
NBFC: A Non-Banking Financial Company (NBFC) in India is a financial institution registered under the Companies Act, 1956/2013, that provides banking-like services such as loans, advances, and investment without holding a full banking license. NBFCs are required to submit various regulatory reports to the Reserve Bank of India. These reports help the regulator monitor financial health, liquidity position, and risk exposure of NBFCs.
However, preparing these reports manually can be time-consuming and prone to errors. This project aims to build a structured understanding of RBI reporting requirements and develop a basic automation tool to simplify report generation.

 3. Objective
The main objectives of this project are:
•	To compile a list of RBI reporting requirements for NBFCs
•	To create a structured catalogue of reports in Excel with details
•	To design sample report formats
•	To develop a prototype tool that converts raw Excel data into RBI reporting formats

 3. Scope of Work
This project is divided into three key parts:
1. Report Catalogue
A comprehensive list of RBI reports was created.

2. Sample Report Formats
Sample templates were created to understand the structure of RBI reports.

3. Prototype Converter Tool
A Python-based tool was developed to automate the conversion process.

Features:
•	Accepts structured Excel input
•	Uses mapping logic to align fields
•	Converts raw data into RBI report format
•	Supports multiple report types (NBS-1 and ALM)
•	Generates formatted Excel output

4. Working of the Tool
The tool works in the following steps:
1.	Reads raw data from Excel input file
2.	Applies mapping rules
3.	Converts data into required report structure
4.	Exports final report in Excel format

5. Dashboard Visualization
A dashboard was created using Power BI Desktop to visualize the reporting data.
This helps in quick analysis and better decision-making.
